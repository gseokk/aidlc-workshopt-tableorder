# Personas

## Persona 1: 일반 고객 (Customer)

| 항목 | 내용 |
|------|------|
| 이름 | 김민준 |
| 역할 | 매장 방문 고객 |
| 연령대 | 20~40대 |
| 기술 수준 | 스마트폰 일상 사용 수준 |

### 특징
- 테이블에 앉아 태블릿/브라우저로 주문
- 별도 회원가입/로그인 없이 즉시 사용 원함
- 메뉴 사진과 가격을 보고 빠르게 선택
- 주문 후 내 주문이 잘 들어갔는지 확인하고 싶음

### 목표
- 직원을 기다리지 않고 원하는 메뉴를 즉시 주문
- 주문 내역과 상태를 언제든 확인

### 불편사항
- 직원 호출 대기 시간
- 주문이 제대로 전달됐는지 불확실함
- 복잡한 앱 설치나 로그인 절차

### 관련 Epic
- Epic 1: 고객 인증 및 세션
- Epic 2: 메뉴 탐색
- Epic 3: 장바구니
- Epic 4: 주문
- Epic 5: 주문 내역 조회

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| 이름 | 이수진 |
| 역할 | 매장 운영자 / 홀 매니저 |
| 연령대 | 30~50대 |
| 기술 수준 | 기본적인 웹 사용 가능 |

### 특징
- 매장 식별자 + 사용자명/비밀번호로 로그인
- 여러 테이블의 주문을 동시에 모니터링
- 주방과 홀 사이에서 빠르게 이동하며 업무
- 테이블 정리 후 새 고객을 위한 세션 초기화 필요

### 목표
- 들어오는 주문을 실시간으로 놓치지 않고 확인
- 테이블별 주문 현황을 한눈에 파악
- 메뉴 변경사항을 즉시 반영
- 테이블 이용 완료 후 깔끔하게 초기화

### 불편사항
- 주문 누락으로 인한 고객 불만
- 수동 계산 오류
- 이전 고객 주문 내역과 현재 주문 혼재

### 관련 Epic
- Epic 6: 관리자 인증
- Epic 7: 실시간 주문 모니터링
- Epic 8: 테이블 관리
- Epic 9: 메뉴 관리
